# 大模型推理评测引擎 - 使用指南

## 📌 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行评测

#### 方式一：使用模拟模式（无需API密钥）
```bash
python main.py --adapter mock
```

#### 方式二：使用OpenAI API
```bash
# 方法1：通过命令行传入API密钥
python main.py --adapter openai --model gpt-3.5-turbo --api-key your-api-key-here

# 方法2：设置环境变量
set OPENAI_API_KEY=your-api-key-here
python main.py --adapter openai --model gpt-3.5-turbo

# 方法3：创建.env文件
# 在.env文件中写入：OPENAI_API_KEY=your-api-key-here
python main.py --adapter openai --model gpt-3.5-turbo
```

#### 方式三：使用Claude API
```bash
# 需要先设置环境变量
set ANTHROPIC_API_KEY=your-api-key-here
python main.py --adapter claude --model claude-3-sonnet-20240229
```

#### 方式四：使用自定义API
编辑 `config.yaml` 文件：
```yaml
model:
  adapter_type: "custom"
  model_name: "your-model-name"
  api_url: "https://your-api-endpoint.com/v1/chat/completions"
  api_key: "your-api-key"
  request_format: "openai"  # 或 "custom"
```

然后运行：
```bash
python main.py
```

## 📊 评测数据说明

当前数据集包含20个样本，涵盖8种任务类型：

| 任务类型 | 样本数 | 说明 |
|---------|--------|------|
| topic_classification | 3 | 主题分类（体育/金融/政治） |
| sentiment_analysis | 3 | 情感分析（正面/中性/负面） |
| entity_type | 3 | 实体类型识别（人物/组织/地点） |
| domain_classification | 3 | 领域分类（医疗/法律/技术） |
| yes_no | 2 | 是非判断 |
| intent_classification | 3 | 意图分类（信息请求/命令/闲聊） |
| polarity_detection | 2 | 极性检测（问题/陈述） |
| robustness_check | 1 | 鲁棒性检查（拼写错误处理） |

## 🔧 自定义数据集

### 数据格式

编辑 `dataset.json` 文件，每个数据项包含：

```json
{
  "id": 1,
  "task_type": "topic_classification",
  "input": "输入文本",
  "classes": ["类别1", "类别2", "类别3"],
  "gold_label": {"label": "类别1"}
}
```

### 支持的标签格式

1. **简单标签**: `{"label": "value"}`
2. **实体+类型**: `{"entity": "name", "type": "type"}`
3. **领域标签**: `{"domain": "domain_name"}`
4. **数值结果**: `{"result": 4}`

## 🚀 高级用法

### 作为Python模块使用

```python
from data_loader import DataLoader
from model_adapters import create_adapter
from inference_engine import InferenceEngine

# 1. 加载数据
loader = DataLoader("dataset.json")
loader.load_data()

# 2. 创建模型适配器
adapter = create_adapter(
    adapter_type="openai",
    model_name="gpt-3.5-turbo",
    api_key="your-api-key"
)

# 3. 创建推理引擎
engine = InferenceEngine(adapter, loader)

# 4. 运行评测
results = engine.run_evaluation(verbose=False)

# 5. 查看结果
print(f"准确率: {results['summary']['总体准确率']}")

# 6. 保存结果
engine.save_results("my_results.json")
```

### 单独评测某一项

```python
# 获取某个数据项
item = loader.get_item(1)

# 单独评测
result = engine.run_single_item(item, verbose=True)

print(f"预测结果: {result['prediction']}")
print(f"是否正确: {result['is_correct']}")
```

### 按任务类型评测

```python
# 只评测情感分析任务
results = engine.run_subset(task_type="sentiment_analysis", verbose=True)
```

## 💡 提示与技巧

### 1. API调用速率限制

如果遇到API速率限制，可以调整 `config.yaml` 中的 `sleep_time`：

```yaml
evaluation:
  sleep_time: 1.0  # 增加到1秒
```

### 2. 节省API费用

- 先用 `--subset` 参数测试少量数据
- 使用 `--task-type` 只评测特定任务
- 先用模拟模式验证流程

### 3. 调试问题

使用 `--verbose` 参数查看详细信息：

```bash
python main.py --adapter openai --verbose --subset 1
```

### 4. 比较多个模型

```bash
# 评测GPT-3.5
python main.py --adapter openai --model gpt-3.5-turbo --output gpt35_results.json

# 评测GPT-4
python main.py --adapter openai --model gpt-4 --output gpt4_results.json

# 评测Claude
python main.py --adapter claude --model claude-3-sonnet-20240229 --output claude_results.json
```

然后比较三个结果文件。

## ❓ 常见问题

### Q: 没有API密钥怎么办？
A: 使用模拟模式：`python main.py --adapter mock`

### Q: 如何添加自己的评测数据？
A: 编辑 `dataset.json`，按照现有格式添加新数据项。

### Q: 支持批量API调用吗？
A: 当前版本是逐条调用，未来版本会添加批量支持。

### Q: 如何修改提示词模板？
A: 编辑 `inference_engine.py` 中的 `build_prompt` 方法。

### Q: 评测结果不理想怎么办？
A: 1) 检查提示词是否清晰；2) 尝试调整温度参数；3) 换用更强大的模型。
