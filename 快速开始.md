# 快速开始指南 ⚡

## 三步开始使用

### 第1步：安装依赖 📦

```bash
pip install -r requirements.txt
```

### 第2步：运行测试 🧪

```bash
python test_engine.py
```

如果看到 "✅ 所有测试通过！"，说明系统运行正常。

### 第3步：开始评测 🚀

#### 选项A：无需API密钥（模拟模式）

```bash
python main.py --adapter mock
```

#### 选项B：使用OpenAI API

```bash
python main.py --adapter openai --model gpt-3.5-turbo --api-key 你的API密钥
```

#### 选项C：使用Claude API

```bash
set ANTHROPIC_API_KEY=你的API密钥
python main.py --adapter claude --model claude-3-sonnet-20240229
```

## 常用命令速查 📋

```bash
# 查看详细评测过程
python main.py --adapter mock --verbose

# 只评测前5条数据
python main.py --adapter mock --subset 1,2,3,4,5

# 只评测情感分析任务
python main.py --adapter mock --task-type sentiment_analysis

# 对比多个模型（需要编辑compare_models.py配置）
python compare_models.py

# 查看帮助信息
python main.py --help
```

## 自定义数据集 ✏️

编辑 `dataset.json`，添加你的评测数据：

```json
{
  "id": 21,
  "task_type": "sentiment_analysis",
  "input": "这个产品真的很棒！",
  "classes": ["positive", "neutral", "negative"],
  "gold_label": {"label": "positive"}
}
```
